Why switch to linux

1. Linux is light and secure

First of all linux is more secure than windows, the reason? because it's open-sources. As windows code is maintained by few devs linux code is updated monthly and has many more developers who work on the kernel. Arch linux fresh install just uses 100Mi to 150Mi memory which compared to windows is way too less, windows 10 fresh install consumes 2Gi of memory even with xorg and dwm the range is around 190Mi to 200Mi on idle.

2. Is freeware

Again not being propreitary like windows.

3. Getting a taste of real computing

Windows and mac were a mistake they made computing really easy which resulted in normies using computers by using linux you get to know how OS's work what is syscall etc, linux promotes terminal use which is really good now you will ask doesn't it make some tasks such as word processing slow ? this is your answer.
